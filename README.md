#How to use
- You must have [Node.js 16.17.1+](https://nodejs.org/en/)
- Install:
```
npm init -y
```
```
npm i disocrd.js
```
```
npm i dotenv
```
- Go to **Package.json** file and type **"type": "module",** in line 4
- Create **.env** file and type:
```
TOKEN=your token here
```
